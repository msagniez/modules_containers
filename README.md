## Repository Structure

Each branch corresponds to an independent module and contains:

- Container definitions
- Workflow definitions
- Module-specific configuration
- CI/CD build pipeline

## Automation

The module status table is maintained automatically by GitHub Actions and updated whenever:

- A module branch receives a new commit.
- A new module branch is created.
- A module branch is removed.
